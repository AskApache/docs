# canonical

`canonical(route = null, params = [])` is a Twig function to create a valid URI to the current page, to use in as an
external link. If a path is given, returns the `canonical` URL for that given path.

For more information,
see [Linking in templates](https://docs.bolt.cm/4.0/templating/linking-in-templates#using-path-and-url-to-link-to-named-routes)
.
