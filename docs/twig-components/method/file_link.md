# file_link

`file_link(line)` is a Twig filter to generate a link to the provided file and line number using a preconfigured scheme.

```twig
{{ file|file_link(line) }}
```

file <br>
type: `string`

line<br>
type: `integer`

Source: [Twig](https://symfony.com/doc/current/reference/twig_reference.html#file-link)
